# Self-hosted fonts (Suknaa web)

These files are committed so `next build` does **not** fetch fonts from Google Fonts or other URLs.

## Tajawal (`tajawal/*.woff2`)

- **License**: SIL Open Font License 1.1  
- **Upstream**: [Tajawal on Google Fonts](https://fonts.google.com/specimen/Tajawal) — source repository [google/fonts OFL/tajawal](https://github.com/google/fonts/tree/main/ofl/tajawal)  
- **Bundled files**: Arabic subset, weights **400**, **500**, **700**, **800** (`*-normal.woff2`).  
- **Snapshot**: Downloaded from npm package `@fontsource/tajawal@5.2.5` files directory (mirrors the Google Fonts release).

## Inter (`inter/*.woff2`)

- **License**: SIL Open Font License 1.1  
- **Upstream**: [Inter on Google Fonts](https://fonts.google.com/specimen/Inter) — source repository [google/fonts OFL/inter](https://github.com/google/fonts/tree/main/ofl/inter)  
- **Bundled file**: Variable Latin `inter-latin-wght-normal.woff2`.  
- **Snapshot**: Downloaded from npm package `@fontsource-variable/inter@5.2.5` files directory.

Full license texts for OFL fonts are available from the upstream repositories above.
