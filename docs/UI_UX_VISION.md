### الخطوط
- **العربية**: Tajawal (Bold للعناوين، Regular للنصوص)
- **الإنجليزية والأرقام**: Inter (Tabular للأسعار)
- **الأرقام دائماً Latin** (1,2,3) حتى في الواجهة العربية

### قواعد ثابتة
- الاتجاه: RTL أولاً (logical CSS: `ms-`, `me-`, `ps-`, `pe-`)
- الظلال: ناعمة ودافئة `rgba(44,40,38, 0.08)`
- الحواف: `8px` صغير، `16px` كروت، `99px` أزرار pill
- لا خلفيات داكنة في صفحات الموقع العام — كل شيء فاتح

## 2. الـ Navbar (مشترك في جميع الصفحات)

```
┌─────────────────────────────────────────────────────────────────┐
│ [سُكنى 🏠]    [الكل●] [عقارات○] [فنادق○]    [AR|EN] [تسجيل الدخول] │
└─────────────────────────────────────────────────────────────────┘
```

**المواصفات التفصيلية:**
- الخلفية: شفافة فوق الـ Hero، تصبح `rgba(255,255,255,0.95)` مع `backdrop-blur: 12px` عند الـ scroll
- الارتفاع: `72px` desktop، `60px` mobile
- اللوجو (يمين): أيقونة بيت برتقالي + نص "سُكنى" بخط Tajawal Bold
- **التابات (وسط)**: ثلاث حبات pill داخل container أبيض بظل خفيف
  - Active Tab: خلفية `#C85A3D`، نص أبيض، ظل `0 4px 12px rgba(200,90,61,0.3)`
  - Inactive Tab: خلفية شفافة، نص `#7A7570`، hover: نص `#C85A3D`
  - تبديل التابات يغير فلاتر الصفحة بدون reload
- زر تسجيل الدخول (يسار): outline style، border `#C85A3D`، نص `#C85A3D`
  - عند الدخول: يتحول لـ Avatar صورة + dropdown
- **Dropdown تسجيل الدخول** (عند الضغط على الزر):
  - "دخول كضيف / مستأجر" → مسار auth guest
  - "دخول كمضيف / مؤجر" → مسار auth host
  - "إنشاء حساب جديد"
- **Mobile**: لوجو + hamburger، التابات تصبح شريط scrollable أسفل الـ nav

## 3. الصفحة الرئيسية (Homepage)

### 3.1 Hero Section
- **الارتفاع**: `100vh` (كامل الشاشة)
- **الخلفية**: صورة سينمائية عالية الجودة لموقع سوري جميل (قلعة الحصن، دمشق القديمة، ساحل اللاذقية) بألوان ذهبية دافئة
- **التدرج**: gradient overlay من الأسفل فقط `linear-gradient(to top, rgba(44,40,38,0.5) 0%, transparent 50%)` — الصورة واضحة وحية
- **العنوان**: مركز، نص أبيض، Tajawal ExtraBold، `font-size: clamp(2.5rem, 5vw, 4rem)`
  - السطر الأول: "سكنك في كل مكان بسوريا"
  - السطر الثاني (أصغر، شفاف): "اكتشف واحجز بيوتاً وفنادق مميزة"

### 3.2 شريط البحث (Search Bar)
- **الشكل**: pill بيضاء عريضة، `border-radius: 99px`، خلفية `#FFFFFF`
- **الظل**: `0 20px 60px rgba(44,40,38,0.15)`
- **الموضع**: مركز الشاشة، 30% من الأسفل، `width: min(780px, 90vw)`
- **الأقسام الداخلية** (مفصولة بخط رمادي فاتح):
  1. 📍 **الوجهة**: placeholder "إلى أين تريد الذهاب؟"
  2. 📅 **التواريخ**: "الوصول — المغادرة"
  3. 👥 **الضيوف**: "عدد الضيوف"
- **زر البحث**: دائرة `#C85A3D`، أيقونة بحث بيضاء، `width: 52px height: 52px`، في نهاية اليسار
- **عند الضغط**: يتوسع كل قسم ليعرض dropdown مناسب

### 3.3 قسم الخريطة (Map Explorer)
- **الخلفية**: `#FBF7F2` (كريمي فاتح)
- **العنوان**: "استكشف على الخريطة" + subtitle فرعي
- **الخريطة**:
  - الحجم: `width: 100%`، `height: 480px`
  - الثيم: MapLibre light style أو OpenStreetMap بـ custom tiles فاتحة (beige/cream)
  - **Markers**: فقاعات سعر ملونة حسب الفئة:
    - 🟧 بيوت/شقق/فيلات → `#C85A3D` (برتقالي)
    - 🟢 مزارع/كبائن/شاليهات → `#3D8A6B` (أخضر)
    - 🟡 فنادق/منتجعات → `#D4A24C` (ذهبي)
    - 🔵 شقق فندقية → `#3D8A95` (تيل)
  - شكل الـ marker: `pill بيضاء` بلون border يطابق الفئة + سعر داخله
  - Hover على marker: mini-card تظهر (صورة + اسم + تقييم + سعر)
- **Filter Sidebar** (يمين الخريطة): أيقونات عمودية (قائمة / خريطة / فلتر)

### 3.4 أبرز الوجهات (Destinations)
- **الخلفية**: `#FFFFFF`
- **العنوان**: "وجهات مميزة" + "عرض الكل →"
- **الكروت**: horizontal scroll، `width: 200px`، `height: 250px`، `border-radius: 16px`
  - صورة كاملة + gradient overlay من الأسفل
  - اسم المدينة بخط أبيض Bold في الأسفل
  - chip صغير "X عقار" بخلفية شفافة
  - Hover: `transform: scale(1.03)` مع transition
- **المدن**: دمشق، اللاذقية، طرطوس، صيدنايا، عجلون، الزبداني

### 3.5 العقارات المميزة (Featured Listings)
- **الخلفية**: `#FBF7F2`
- **العنوان**: "أماكن مميزة" + tab filter [الكل | عقارات | فنادق]
- **الشبكة (Bento Grid)**:
  - كارت Hero كبير: `col-span: 2، row-span: 2` (العقار الأجمل)
  - 4 كروت صغيرة على الجانب
- **مواصفات كارت العقار**:
  - صورة بنسبة 16:10 مع `border-radius: 16px`
  - زر ❤️ (wishlist) top-left مع خلفية شفافة
  - badge نوع العقار top-right (لون الفئة)
  - اسم العقار + موقع
  - ⭐ تقييم (أرقام Latin) + عدد التقييمات
  - سعر الليلة في `pill #C85A3D` + زر "احجز"
  - Shadow: `0 8px 24px rgba(44,40,38,0.08)`
  - Hover: shadow يكبر + رفع خفيف `transform: translateY(-4px)`

### 3.6 لماذا سُكنى؟ (Why Suknaa Strip)
> **القرار (جلسة 4)**: تم **إزالة** "شريط أعلن عن بيتك" نهائياً من الصفحة الرئيسية.
> الزوار الـ 90% هم باحثين عن سكن — لا نضغط عليهم بإعلان استضافة.
> الاستضافة لها صفحة كاملة مستقلة `/host` (انظر القسم 4) + دعوة لطيفة في الفوتر فقط.

- **الشكل**: شريط أفقي عريض، خلفية `#FFFFFF`، حدود علوية وسفلية رفيعة `#F5EFE6`
- **الارتفاع**: `200px` desktop، `auto` mobile
- **المحتوى**: 4 ميزات أفقية بأيقونات مخصصة (لـيست emoji عادية):
  | الأيقونة | العنوان | الوصف الفرعي |
  |---|---|---|
  | 🛡️ Custom Shield | حجز موثّق ومضمون | نتحقق من كل عقار قبل النشر |
  | 💳 Wallet Icon | الدفع آمن 100% | لا تدفع للمضيف مباشرة — Sham Cash, MTN, تحويل |
  | ⭐ Star with sparkle | تقييمات حقيقية | كل مراجعة من ضيف حجز فعلاً |
  | 📞 Headset Icon | دعم 24/7 بالعربي | WhatsApp + Email سريع |
- **النص**: عنوان `Tajawal Bold 1.125rem #2C2826` + subtitle `Tajawal Regular 0.875rem #7A7570`
- **التحريك**: عند الـ scroll، الأيقونات تظهر بـ `stagger fade-in` كل 100ms
- **Mobile**: عمود واحد، الأيقونات يمين والنص يسار (RTL)

### 3.7 الوجهات المميزة الموسمية (Seasonal Picks) — *جديد إبداعي*
- **الفكرة**: قسم ديناميكي يتغير حسب الموسم (صيف → ساحل، شتاء → جبال، رمضان → دمشق القديمة)
- **الخلفية**: `#FBF7F2` (كريمي)
- **العنوان**: "🌊 وجهات الصيف المختارة" أو "❄️ ملاذات الشتاء" (يتغير ديناميكياً من backend)
- **الكروت**: 3 كروت أكبر من قسم الوجهات العادي، نسبة 4:5 عمودية، `width: 320px`
  - صورة سينمائية + gradient overlay من الأسفل
  - chip في الأعلى: "🔥 الأكثر طلباً هذا الأسبوع" أو "✨ مكتشف جديد"
  - عنوان كبير + 2-3 سطور وصف شعري قصير
  - زر "اكتشف →" بـ ghost style أبيض شفاف
- **القاعدة الذهبية**: لا تظهر أبداً بدون بيانات حقيقية. إذا لا توجد بيانات موسمية، يُخفى القسم.

### 3.9 المدخل الذكي للمضيفين (Smart Host Entry) — *جديد*
- **القاعدة**: لا banner ضاغط في الصفحة الرئيسية. بدلاً من ذلك:
  - **في الـ Navbar dropdown** (عند الضغط على "تسجيل الدخول"): خيار "دخول كمضيف" + خيار "هل أنت مضيف جديد؟ ابدأ الآن" → يقود إلى `/host`
  - **في الفوتر**: عمود "للمضيفين" واضح وصريح
  - **في صفحة `/host`**: pitch كامل لإقناع المضيف (انظر القسم 4)
  - **بعد تسجيل الدخول كمضيف لأول مرة**: empty state في dashboard فيه CTA كبير "أضف عقارك الأول" (انظر القسم 9)
- **المنطق**: 90% من زوار الموقع باحثون. شريط ضاغط في الهوم لكل زائر = ضوضاء بصرية. الـ 10% المؤجرون يصلون عبر مسار واضح ومخصص.

## 4. صفحة "كن مضيفاً" (Host Landing — `/host`) — *جديد كاملاً*

> **الغرض**: صفحة منفصلة 100% للإقناع وجذب المضيفين الجدد.
> هذه الصفحة هي البديل الذكي عن الـ banner المُزال من الصفحة الرئيسية.
> الزائر يصل لهنا عبر: الـ navbar dropdown، الفوتر، أو روابط مباشرة من حملات تسويقية.

### 4.1 Hero Section (Host)
- **الارتفاع**: `90vh`
- **الخلفية**: صورة سينمائية لـ مضيف سوري حقيقي يستقبل ضيوف في بيته الدمشقي/الشامي (إضاءة ذهبية دافئة)
- **التدرج**: gradient overlay من اليسار `linear-gradient(to right, rgba(44,40,38,0.7) 0%, rgba(44,40,38,0.3) 60%, transparent 100%)`
- **المحتوى** (يمين، RTL):
  - عنوان: "**حوّل عقارك إلى مصدر دخل**" — Tajawal ExtraBold `clamp(2.5rem, 5vw, 4rem)` أبيض
  - subtitle: "انضم إلى مجتمع المضيفين الموثوقين في سوريا. مع سُكنى، أنت تتحكم بكل شيء — السعر، التوفر، القواعد."
  - زرّان متجاوران:
    - **الأساسي**: "ابدأ الاستضافة الآن →" (`#C85A3D` أبيض، pill كبير)
    - **الثانوي**: "كيف تعمل المنصة؟" (ghost border أبيض)

### 4.2 الأرقام التي تتحدث (Live Numbers Strip)
- شريط أفقي على خلفية `#FBF7F2`
- 4 أرقام كبيرة بـ Tabular Inter Bold:
  - **1,200+** عقار مُسجَّل
  - **15** مدينة في سوريا
  - **4.7⭐** متوسط التقييم
  - **24h** وقت الموافقة
- *ملاحظة*: تظهر فقط عند توفر بيانات حقيقية. لا أرقام وهمية.

### 4.3 لماذا سُكنى للمضيف؟ (3-Column Benefits)
- 3 كروت كبيرة، خلفية بيضاء، ظلال دافئة:
  | العنوان | الوصف |
  |---|---|
  | 💰 **اختر طريقة العمولة** | العمولة مرنة حسب حسابك/عقدك. اختر تتحملها أنت أو تمررها ضمن السعر الظاهر — التحكم لك. |
  | 🛡️ **حماية كاملة** | نتحقق من كل ضيف. الحجوزات مدفوعة قبل الوصول. لا متأخرات، لا عمليات احتيال. |
  | 📞 **دعم بالعربي** | فريق سوري يفهمك. WhatsApp مباشر، email، وقت استجابة < 30 دقيقة. |

### 4.4 كيف تعمل سُكنى؟ (How It Works — 4 Steps)
- شريط أفقي بـ 4 خطوات مرتبطة بخط نقطي:
  1. **🏠 أضف عقارك** — صور + وصف + مميزات (15 دقيقة)
  2. **✓ نتحقق منك** — KYC سريع (24-48 ساعة)
  3. **📅 افتح التقويم** — تحدد أنت متى متاح ومتى لا
  4. **💵 استقبل الضيوف** — الدفع مضمون، المحفظة تنمو
- كل خطوة كارت مستقل بأيقونة كبيرة + رقم في زاوية + خط واصل بين الكروت

### 4.5 آلة حاسبة الدخل التقديري (Earnings Calculator) — *مفاجأة إبداعية*
- **الفكرة**: صندوق تفاعلي يقدّر دخل المضيف الشهري قبل ما يسجّل
- **الإدخال**:
  - نوع العقار (dropdown): بيت / شقة / فيلا / مزرعة / شاليه...
  - المدينة (dropdown): دمشق / اللاذقية / طرطوس...
  - عدد الغرف (stepper)
- **الإخراج** (يحدّث live):
  - **سعر مقترح/ليلة**: $X (مبني على بيانات السوق)
  - **معدل الإشغال المتوقع**: Y%
  - **الدخل الشهري التقديري**: $Z (بعد العمولة)
  - زر CTA: "ابدأ بـ هذا السعر →"
- *المنطق التقني*: backend module `pricing-suggestions` يعطي الأرقام (Phase 8، أو median بسيط من بيانات السوق في Phase 1)

### 4.6 شهادات المضيفين (Host Testimonials)
- 3-4 شهادات حقيقية، carousel أفقي
- كل شهادة: صورة المضيف + اسم + مدينة + اقتباس مباشر + إحصائية واحدة (مثال: "$2,400 شهرياً")
- خلفية `#FBF7F2`

### 4.7 الأسئلة الشائعة للمضيفين (Host FAQ)

- أسئلة: العمولة كم؟ متى استلم فلوسي؟ ماذا لو ألغى الضيف؟ الـ KYC شو يحتاج؟ هل أقدر إيقاف عقاري مؤقتاً؟ ...
- 8-10 أسئلة كأقصى حد

### 4.8 CTA النهائي
- شريط full-width بـ gradient `linear-gradient(135deg, #C85A3D 0%, #D4A24C 100%)`
- "**جاهز تبدأ؟**" + زر أبيض "سجّل كمضيف الآن →"

5. **المميزات والخدمات** — checkboxes منظمة بفئات (الأساسيات، المطبخ، الترفيه، الأمان...)
6. **الصور** — drag & drop، minimum 5، تحديد صورة الغلاف، إعادة ترتيب
7. **التسعير** — 4 tiers + Commission Choice Toggle (نفس الـ component في DESIGN_SYSTEM §7.10)
8. **المراجعة والنشر** — preview كامل + checkbox "أوافق على شروط الاستضافة" + زر "نشر الآن" أو "حفظ كمسودّة"

**Auto-save**: كل تغيير يُحفظ تلقائياً بـ debounce 1 ثانية، indicator "✓ تم الحفظ"
