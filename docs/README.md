## 🆕 ما الجديد في v2؟

هاي النسخة تعكس قرارات معمارية كبيرة وضّحناها مع بعض. التغييرات الرئيسية:

### 1. **نظامان منفصلان كلياً**
- **نظام عقارات (Real Estate / P2P)**: بيوت / شقق / فيلات / مزارع / كبائن / شاليهات / استوديوهات
- **نظام فنادق (Hospitality / B2B)**: فنادق / شقق فندقية / منتجعات / نُزُل
- كل نظام له جداول DB منفصلة، endpoints منفصلة، modules منفصلة

### 2. **واجهة موحّدة بثلاثة tabs**
- شريط دائم فوق: `[الكل] [عقارات] [فنادق]`
- الافتراضي: "الكل"
- يبدّل بين النظامين بسلاسة

### 3. **زرّان منفصلان لتسجيل الدخول**
- "دخول كزبون" / "دخول كمؤجر"
- نفس الحساب يمكن أن يكون الاثنين

### 4. **نظام تسعير بـ 4 طبقات**
- سعر الليلة الأساسي
- سعر الليلة لو الحجز ≥ 7 ليالي (أسبوعي)
- سعر الليلة لو الحجز ≥ 30 ليلة (شهري)
- أسعار موسمية (صيف، عيد، رمضان، إلخ)
- + زيادة عطلة نهاية الأسبوع اختيارية

### 5. **العمولة الذكية (Commission Passthrough)**
المؤجر يختار:
- **"العمولة عليّ"** (الافتراضي): يكتب $50 → يستلم $44 → الزبون يشوف $50
- **"تمرير على الزبون"**: يكتب $50 صافي → النظام يحسب $56.82 → الزبون يشوف $56.82

**في كل الحالات:** الزبون لا يرى ذكر العمولة في الفاتورة. يشوف فقط: سعر العقار + رسوم خدمة/ضرائب قانونية عند وجودها.

### 6. **رسوم الخدمة منفصلة عن العمولة**
- العمولة قابلة للإلغاء أو التعديل (محتفز للمؤجرين الجدد)
- رسوم الخدمة تظهر للزبون كبند منفصل عندما تكون مطبقة
- النظام يقدر يعمل عروض ترويجية مرنة

### 6.1. **قواعد مالية مرنة وليست أرقام ثابتة**
- لا توجد عمولة أو رسوم خدمة أو ضرائب ثابتة داخل الكود
- الأرقام الافتراضية مجرد seed/config كبداية، ويمكن تعديلها حسب المؤجر/الفندق/العقار/الغرفة/الزبون/العرض
- الفندق أو الشركة يمكنه إدخال الضرائب/الرسوم المحلية، وسُكنى تملك صلاحية الاعتماد أو التعديل أو التعطيل
- كل حجز يخزن snapshot للقواعد المالية التي طُبّقت عليه حتى لا تتغير الفواتير القديمة عند تعديل الإعدادات لاحقاً

### 7. **نظام تسجيل البيوت = Airbnb**
- كل غرفة تتوصف لحالها (نوع السرير، صور، مميزات)
- الحمامات (عدد + نوع)
- المطبخ، صالون، حديقة، إلخ — كل واحد بصور ومميزات
- مميزات البيت ككل (مسبح، موقف، إلخ)

### 8. **نظام تسجيل الفنادق = Booking.com**
- بيانات الفندق (اسم، عدد نجوم، وصف)
- صور للفندق ككل (مصنفة: لوبي، مطعم، مسبح، إلخ)
- خدمات فندقية (24h reception، إفطار، سبا)
- سياسات (أطفال، حيوانات، تدخين)
- **نظام أنواع الغرف**: كل نوع غرفة لحاله (سعر، عدد، مميزات، صور)
- **تتبع الوحدات الفعلية**: 5 غرف دبل = 5 سجلات منفصلة في `room_units`

### 9. **خرائط بألوان مميزة**
- 🟧 برتقالي → بيوت
- 🟡 ذهبي → فنادق
- 🟢 أخضر → مزارع/كبائن/شاليهات
- 🔵 فيروزي → شقق فندقية

### 10. **الأماكن السياحية القريبة على الخريطة**
- مطاعم، أسواق، مساجد، مستشفيات، مواقع سياحية
- كل واحد مع المسافة (مشي/سيارة)
- مقدّر للعرض ضمن دائرة 2 كم حول العقار

### 11. **نظام الذكاء الاصطناعي للأسعار (Price Intelligence)**
- **للمؤجر**: تنبيهات السوق ("الطلب في دمشق ارتفع 30%، انصح برفع السعر")
- **عند تسجيل عقار**: مساعد التسعير ("سعرك مرتفع/منخفض مقارنة بعقارات مشابهة")
- المنصة تجمع snapshots يومية للسوق وتولد اقتراحات مخصصة

### 12. **حماية من التأجير خارج المنصة (Anti-Circumvention)**
- لما المؤجر يخفض المتاح، يلزمه يحدد سبب
- نظام تقييم مخاطر تلقائي (Risk Score)
- 4 مستويات: Low / Medium / High / Critical
- مستويات Medium وأعلى تنبه الإدمن للمراجعة
- العقوبات تدريجية، لا حظر تلقائي

### 13. **تنبيهات الندرة (Scarcity Nudges)**
- "متبقي غرفة وحدة بهذا السعر!"
- "تم حجز 3 غرف بآخر 24 ساعة"
- "أفضل سعر بهذه المنطقة"
- **فقط بناءً على بيانات حقيقية** — لا تنبيهات وهمية

### 14. **نظام Upgrade ذكي**
- "بـ $15 إضافية، احصل على غرفة أفضل"
- يعرض عند صفحة الحجز إذا تتوفر غرفة أحسن

### 15. **تقييم مزدوج (Dual Reviews)**
- تقييم العقار/الفندق (نظافة، دقة، موقع، قيمة، مرافق)
- تقييم المضيف (تواصل، استجابة، ضيافة)
- متوسط الاثنين يظهر على الصفحة

### 16. **صفحات للمضيفين (Host Profiles)**
- معلومات + تقييم متوسط + كل عقاراته
- زبون يضغط على اسم المضيف ويشوف تاريخه

### 17. **ميزات إضافية للزبون**
- **Wishlist مع مشاركة** (أرسل قائمتك لعيلتك)
- **مقارنة عقارات** (4 عقارات بنفس الوقت)
- **تنبيه السعر** (خبرني لو نزل السعر)

### 18. **أنواع المضيفين الجديدة**
- **نظام عقارات (P2P)**: شخص فردي / مكتب عقاري
- **نظام فنادق (B2B)**: شركة فندقية فقط
- KYC مختلف لكل نوع (مكاتب وفنادق تحتاج رخصة تجارية)

## ✅ خطوات الانطلاق (Phase 0)

1. **VPS**: احجز Hostinger VPS KVM 2 (~$7/شهر) أو KVM 4 (~$10/شهر)
   - Datacenter: Frankfurt أو Amsterdam
   - OS: Ubuntu 24.04 LTS (لا تختار NodeJS App template)
2. **DNS**: ربط `suknaa.com` بـ Cloudflare
3. **Repo**: مستودع GitHub خاص (Private)
4. **Docs**: ضع هاي الوثائق في `docs/` بالـ repo
5. **Cursorrules**: ضع `.cursorrules` بجذر الـ repo
6. **Mockups**: 5 صفحات على Figma أو v0.dev:
   - الرئيسية (مع الـ tabs الـ 3 + زرّي تسجيل الدخول)
   - نتائج بحث (عقارات)
   - صفحة فندق (مع room types)
   - صفحة بيت (مع per-room sections)
   - واجهة Commission Passthrough Toggle (مهم جداً)
7. **شارك** Mockups مع 5 أصدقاء واطلب رأيهم
8. **عدل** قبل ما تبدأ بالكود
